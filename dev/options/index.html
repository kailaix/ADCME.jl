ADCME.reset_default_options — Function
reset_default_options()

Results the ADCME options to default.

source

The current options and their default values are

using PrettyPrint
using ADCME
pprint(ADCME.options)
ADCME.Options(
  sparse=ADCME.OptionsSparse(
    auto_reorder=true,
    solver="SparseLU",
  ),
  newton_raphson=ADCME.OptionsNewtonRaphson(
    max_iter=100,
    verbose=false,
    rtol=1.0e-12,
    tol=1.0e-12,
    LM=0.0,
    linesearch=false,
    linesearch_options=ADCME.OptionsNewtonRaphson_LineSearch(
      c1=0.0001,
      ρ_hi=0.5,
      ρ_lo=0.1,
      iterations=1000,
      maxstep=9999999,
      αinitial=1.0,
    ),
  ),
)
